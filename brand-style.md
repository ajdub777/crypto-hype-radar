# Crypto Hype Radar — Brand & Style Guide

This document defines the visual identity, tone, and design rules for Crypto Hype Radar. It serves as the source of truth for all product UI, social media content (Instagram, X, Reddit), and marketing materials.

## 1. Core Aesthetic
- **Theme**: Dark Cyberpunk / Neon Tech
- **Vibe**: High-signal, data-driven, urgent but clean, futuristic but accessible.
- **Backgrounds**: Deep blacks and very dark grays. Never pure white.
- **Accents**: High-contrast neon glows (Cyan, Green, Red, Purple).

## 3. Typography
- **Primary Font**: Inter (or system sans-serif like San Francisco/Roboto).
- **Style**: Clean, highly legible, modern.
- **Weights**:
  - **Bold (700)**: Coin tickers, major data points (e.g., "78", "+15%").
  - **Medium (500)**: Section headers, button text.
  - **Regular (400)**: Body text, timestamps.

## 4. UI & Layout Rules
- **Borders**: Thin, subtle borders (`border-neutral-800`) to separate elements.
- **Glow Effects**: Use subtle drop shadows with neon colors for emphasis (e.g., `shadow-[0_0_15px_rgba(6,182,212,0.5)]` for cyan glow).
- **Corners**: Rounded corners (`rounded-xl` or `rounded-2xl`) for cards and buttons to soften the harsh dark theme.
- **Spacing**: Generous padding inside cards, tight grouping of related data points. Mobile-first proportions.

## 5. Social Media Content (Instagram / X)
When generating social media posts (like the Daily Top Movers), adhere to these rules:
- **Format**: 9:16 ratio for Instagram Reels/Stories, 4:5 or 1:1 for feed posts.
- **Background**: Dark, subtle tech/circuit board textures or pure deep gray.
- **Layout**:
  - **Header**: Clear, bold title (e.g., "DAILY TOP MOVERS").
  - **Data Rows**: Coin Logo + Ticker + 24h % Change + Hype Score.
  - **Visuals**: Include the Fear & Greed thermometer as a prominent visual element.
  - **Footer**: Website URL (`cryptohyperadar.com`) and a subtle call to action.
- **Tone**: Objective, data-focused, slightly hype but not scammy. "The data speaks for itself."

## 6. Voice & Tone
- **Direct**: "Bitcoin is up 5%." Not "It looks like Bitcoin might be having a good day."
- **Data-Backed**: Always reference the Hype Score or Fear & Greed index.
- **Action-Oriented**: Use clear verbs (Trade, Watch, Share).
- **No Financial Advice**: Always maintain the stance that this is a data tool, not a financial advisor.
